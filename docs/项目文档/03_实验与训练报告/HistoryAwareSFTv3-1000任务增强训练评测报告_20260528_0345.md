## 结论

本阶段完成了 1000 个本地 browser GUI-RL 任务、坐标扰动增强、失败恢复示范、history-aware SFT 训练和独立 test100 评测。

这里的 `SFT` 指监督微调，用 scripted oracle 的专家轨迹让本地模型先学会看截图并输出 direct GUI action JSON；`rollout` 指一次完整任务尝试轨迹，从 reset 后多步执行到成功、失败或超时；`verifier` 指验证器，用程序检查任务是否完成并返回 reward。

- v3 1000 任务增强版 test100：success_rate=0.82

- valid_action_rate=1.0
- policy_error_rate=0.0

这说明本地 Qwen2.5-VL-3B 已经具备较稳定的 browser GUI 操作 warmup 能力，但这仍不是严格 on-policy RL。严格 `on-policy RL` 指在线策略强化学习，即当前模型自己产生 rollout，再用 verifier reward 更新当前模型。下一步可以开始小规模 verifier-guided RL。
