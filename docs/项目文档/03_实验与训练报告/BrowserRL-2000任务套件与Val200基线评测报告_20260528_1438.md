# BrowserRL 2000 任务套件与 Val200 基线评测报告

- 生成时间：2026-05-28 14:38 CST
- 主线：为 verifier-guided on-policy VLM agentic RL 建立更可靠的 train/val/test 基准。
- 结论：2000 tasks 已构建，oracle center 校验 2000/2000 成功；当前 v3 SFT baseline 在新 `val200` 上 success_rate=0.745。后续 RL 应以这个 `val200` 作为正式 evaluation gate。

## 术语说明

- `task suite`：一组可 reset、可执行、可 verifier 自动判分的浏览器 GUI 任务。
- `val`：验证集，用于训练过程中选模型、早停和回滚。
- `test`：测试集，用于最终验收，不能频繁拿来调参。
- `oracle`：脚本化专家策略，直接用 selector 执行正确动作，用来验证任务是否可解。
- `family-balanced`：按任务大类均衡抽样，避免评测只覆盖某几个类型。

## 新增和修改

新增脚本：

- `scripts/build_browser_rl_eval_subset.py`
  - 从 val/test 中按 family 均衡抽样。
  - family 内尽量按 template 均衡，例如 choice 会覆盖 checkbox/radio/select。

修改脚本：

- `scripts/build_browser_rl_task_suite.py`
  - 修复切分逻辑：从 family-level split 改成 template-level split。
  - 原因：旧切分会导致 choice 的 val/test 只出现 `choice_select`，advanced 的 val/test 只出现 `advanced_tab`，评测不可靠。

- `scripts/train_browser_rl_onpolicy_grpo.py`
  - 增加 `--family-quotas-json`。
  - 增加 `--target-trainable-groups`。
  - 增加 `--dedupe-actions` 和 `--max-sample-attempts`。
  - 增加 `--logprob-reduction mean`，降低 JSON 长度带来的梯度放大。

## 主要命令

构建 2000 tasks：

```bash
/opt/conda/envs/llama/bin/python3 scripts/build_browser_rl_task_suite.py \
  --output-dir outputs/browser_rl_task_suite_2000_20260528_1344 \
  --timestamp 20260528_1344 \
  --seed-offset 0 \
  --train-ratio 0.8 \
  --val-ratio 0.1 \
  --counts-json '{"form":450,"search":400,"choice":350,"menu":250,"table":250,"todo":200,"advanced":100}'
```

构建 balanced eval：

```bash
/opt/conda/envs/llama/bin/python3 scripts/build_browser_rl_eval_subset.py \
  --input outputs/browser_rl_task_suite_2000_20260528_1344/val_tasks.jsonl \
  --output outputs/browser_rl_task_suite_2000_20260528_1344/val_balanced_70_tasks.jsonl \
  --per-family 10 \
  --seed 1344
```

val200 baseline 双卡评测：

```bash
CUDA_VISIBLE_DEVICES=0 /opt/conda/envs/llama/bin/python3 scripts/run_browser_rl_rollouts.py \
  --env-source local_smoke \
  --policy local_qwen \
  --tasks outputs/browser_rl_task_suite_2000_20260528_1344/val_shards/val_00.jsonl \
  --output-dir outputs/rollouts_local_qwen25vl_history_aware_v3_2000_val200_20260528_1412/shard_00 \
  --max-steps 6 \
  --headless \
  --local-model /root/models/Qwen2.5-VL-3B-Instruct \

  --local-load-in-4bit \
```
