## 数据目录迁移

`BrowserRL`：浏览器强化学习环境，指用本地网页任务、Playwright 执行器和 verifier reward 组成的可 reset/step/observe/reward 环境。

本次建立了正式数据目录：

```text
/root/datasets/browser_rl/
├── task_suites/
├── sft/
└── onpolicy/
```

其中：

- `task_suites/`：固定任务套件。
- `sft/`：监督微调数据。SFT 是 Supervised Fine-Tuning，中文是监督微调，用专家轨迹训练模型先学会合法动作格式和基本操作。
- `onpolicy/`：在线策略采集数据。On-policy 指当前模型自己 rollout 后产生的数据，而不是静态专家数据。
